De begeleiding

Vragen/problemen

Bij begeleiding worden we begeleid bij dingen die we niet weten.
Als we vragen hebben of iets anders kunnen we naar onze SLB'er gaan.
Ook als er problemen zijn bijvoorbeeld kunnen we het aan iemand vertellen die er iets van weet.

SLB'er

Een SLB'er is een persoon zoals een mentor waar je heen kunt gaan als je vragen of problemen hebt.
Of als er iets anders is kan je ook naar hun gaan.

De software

De software die we gebruiken

Wij gebruiken verschillende software bij elk vak.
Bijvoorbeeld bij Nederlands en engels gebruiken we verschillende licenties dan bij programmeren en het project wat we nu maken.
Elke licentie heeft daarom ook een andere taak.
Bij de meeste lessen gebruiken we Visual Studio Code waarin we onze opdrachten maken.
Maar we gebruiken Github om alles op te slaan in het geval dat onze laptop crashd of zomaar uitgaat.
